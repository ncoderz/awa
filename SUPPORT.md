# Support

## Getting Help

- Read [README.md](README.md)
- Review [docs/CLI.md](docs/CLI.md), [docs/TEMPLATE_ENGINE.md](docs/TEMPLATE_ENGINE.md), and [docs/WORKFLOW.md](docs/WORKFLOW.md)
- Search existing issues before opening a new one

## Bug Reports

Use the GitHub issue tracker with a clear reproduction.

Include:

- Node.js version
- Command used
- Expected behavior
- Actual behavior
- Relevant logs or output

## Feature Requests

Open an issue describing:

- The problem you are trying to solve
- The proposed behavior
- Any alternatives you considered

## Security Issues

For vulnerabilities, follow [SECURITY.md](SECURITY.md) and report privately.
