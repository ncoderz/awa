# ARCHITECTURE

## Purpose

This is a test workspace for the awa VS Code extension. It contains minimal spec
and source files to exercise all LSP features: hover, go-to-definition,
diagnostics, autocomplete, code lens, and rename.

## Components

| Component | Purpose |
| --------- | ------- |
| `EX-Parser` | Parses input data |
| `EX-Formatter` | Formats output data |
